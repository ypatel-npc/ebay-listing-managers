Bulk Upload Listings

CSV File Requirements:

Your CSV file must include the following columns:

  • SKU - Your unique identifier for the product
  • Localized For - Locale information (e.g., en_US)
  • title - Product title (80 characters max)
  • subtitle - Optional subtitle for the listing
  • description - Product description (HTML allowed)
  • Picture URL 1 - URL to product image
  • MPN - Manufacturer Part Number
  • Brand - Brand name of the product
  • Condition - Item condition (e.g., New, Used)
  • Condition Description - Details about the condition
  • Compatible Product 1 - Compatible product information
  • Total Ship to Home Quantity - Available quantity
  • LocationId - Location identifier
  • Fulfillment Type - How the item will be fulfilled
  • Quantity Available - Available quantity per channel
  • Channel ID - Sales channel identifier
  • Category - eBay category ID
  • Secondary Category - Additional category ID
  • Shipping Policy - Shipping policy name/ID
  • Payment Policy - Payment policy name/ID
  • Return Policy - Return policy name/ID
  • List Price - Product price
  • Apply Tax - Whether to apply tax (Yes/No)
  • Warehouse Location ID - Warehouse identifier

Attribute fields:

  • Attribute Name 1-9 - Names of item specific attributes
  • Attribute Value 1-9 - Values for the corresponding attributes
